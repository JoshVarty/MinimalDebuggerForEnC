ReadMe.txt for raw project.

The files in this project should not be modified.  They represent a COM marshaling code used by the sample. COM marshaling code is used to call 
native Managed Debugging COM API functions.

The functions available in this dll should primarily be called from the corapi Managed Wrappers.  See that ReadMe for details.
